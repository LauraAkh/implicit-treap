# Неявное декартово дерево с отложенными операциями

Шаблонная реализация неявного декартова дерева (рандомизированного бинарного дерева поиска) на C++. Курсовая работа по объектно-ориентированному программированию, РТУ МИРЭА.

## Технологии
- C++17, шаблоны
- ООП (классы `node`, `treap`, `iterator`)

## Возможности
- **Базовые операции:** merge, split (за O(log n))
- **Запросы на отрезке:** сумма, максимум
- **Обновления:** переворот отрезка (с отложенным выполнением)
- **Два режима:**
  - Быстрый (разрушающий) – модифицирует исходные деревья
  - Безопасный (копирующий) – операнды не меняются
- Собственные исключения для обработки ошибок

## Пример использования
```cpp
#include "implicit_treap.hpp"

int main() {
    imptreap<int, double> tree;
    tree.merge(1, 1.5);   // вставка элемента (индекс, значение)
    tree.merge(2, 5.5);
    // ... ещё вставки
    std::cout << tree.find_product(2, 5); // произведение на отрезке [2,5]
    return 0;
}
